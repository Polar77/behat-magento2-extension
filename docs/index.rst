Behat Magento 2 Extension
=========================

This Behat extension provides provides a custom service container for Behat which allows to inject Magento services into Behat Contexts and Behat helper services.
The module ships some default actions, but users are highly encouraged to create their own logic.

This module is tested for Magento 2.4.x and PHP 7.4 and above. In the future, support for PHP7.4 will be dropped.
